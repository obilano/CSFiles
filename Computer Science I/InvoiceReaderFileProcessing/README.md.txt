Write a C++ program to read invoice data from a file with the following format:

invoiceID productID quantity unitCost

For each line of input, create an invoice that looks like this (no blank lines in output):

======================================

Invoice #100036

Product ID: 444

Quantity: 2

Unit Cost: $4.99

Subtotal: $9.98

Tax: $0.85

Total: $10.83

=======================================
